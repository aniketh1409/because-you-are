# Because You Are (Work in Progress)
## About the Project
Because You Are is a short, linear interactive experience built in Godot 4. Itâ€™s inspired by Undertale and focuses on a 5 -15 minute emotional journey about self-love. The goal is to walk through a series of "frames," interacting with NPCs who reflect your value back to you through dialogue and small tasks.

## Tech & Assets
Engine: Godot 4.

Art: 16x16 top-down pixel art assets.

Style: Top-down RPG perspective with custom UI layering and fade transitions.

### Current Progress (What's Actually Working)

Movement System: Working top-down player controller with speed/accel/friction and movement gating.

Interaction Logic: Area2D proximity detection and "Press Enter to talk" prompts.

Dialogue System: Functional UI with NPC text labels and choice-based button responses.

World Setup: Beach tileset implemented with sand and palm tree objects.

Screen Effects: Smooth fade-in/fade-out transitions for scene intros.
